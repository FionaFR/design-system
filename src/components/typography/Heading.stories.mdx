# Headings

`<Heading>`, `<Heading2>`, `<Heading3>` components should be used instead of `<h1>`, `<h2>` and `<h3>`.

<Preview>
  <Story name="All Headings">
    <>
      <Heading>Heading</Heading>
      <Heading2>Heading2</Heading2>
      <Heading3>Heading3 (we recommend not going smaller than this)</Heading3>
    </>
  </Story>
</Preview>

## Custom style

The component behaves just like a normal heading tag. Just use the `style` attribute if you need modifications.

<Preview>
  <Story name="Custom Styled">
    <>
      <Heading style={{ color: 'red' }}>Heading</Heading>
    </>
  </Story>
</Preview>
