# How to use this component

## Choosing an icon

This icon-component uses the material-icon fonts from google.

You can find all the different icons (and the different types of icons, rounded, sharp etc.) [here](https://material.io/resources/icons/?style=baseline).

You set the type of icon you want to use by passing the `type` prop to the component. The different types are baseline (this is the default, so no prop is needed), `outline`, `two-tone`, `round` or `sharp`.

## Styling the icon

To change the size of the icon pass the size prop to the icon.

This prop can either be small, medium, large or huge. You can also change the font-size later in a custom-css document by changing the font-size of the `<i>` element.

To change the color of the icon, pass the color prop. This just needs to be any valid CSS color. (Hex, RGB, etc.)
